# 🔐 Bcrypt Hash Generator Extension

A lightweight browser extension to generate **bcrypt hashes locally** with a configurable cost factor (weightage).

No data collection. No tracking. No external requests.

---

## ✨ Features
- Generate bcrypt hashes instantly
- Adjustable cost factor (default: **12 – recommended**)
- Fully client-side (uses bcryptjs)
- One-click copy to clipboard
- Clean and simple UI
- Laravel / PHP compatible output

---

## 🧠 How it works
All hashing is done **locally in your browser**.  
Your input never leaves your machine.

---

## 🧪 Usage
1. Enter the string you want to hash
2. Select cost factor (weightage)
3. Click **Generate Hash**
4. Copy and use the bcrypt value

---

## 🛡 Privacy
This extension does **not**:
- Collect user data
- Track activity
- Make network requests
- Store inputs or hashes

---

## 🧰 Tech Stack
- JavaScript
- bcryptjs
- Chrome Extension (Manifest V3)

---

## 📦 Installation (Developer Mode)
1. Clone this repository
2. Open `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder
